PuppetMaster by garaud

Provides facilities to deal with computations on a Linux network.
  • Get some useful information about a network of hosts such as:
    • number of cores
    • system load averages
    • total system memory
    • current used memory
    • available hosts with respect to the load and the number of cores
  • Take advantage of multi-threading to get some network information faster.
  • Launch several different commands on a network automatically (via SSH)

Install

You must have Python (official website). This project was tested with the Python version 2.5.2
Download the project and go to a directory which is specified in the PYTHONPATH shell variable. Then, just do 

$ ln -s /path/to/puppetmaster/src puppetmaster
You can import the module in a Python script with from puppetmaster import *

Quickstart

In a Python environment, do 

     >>> from puppetmaster import *
     >>> h = Host()
     >>> h.GetProcessorNumber()
     2
     >>>
     >>> h.GetUptime()
     [0.102, 0.12, 0.089]
     >>>
     >>> h.LaunchFG('pwd')
     (0, '/home/login/puppetmaster/src')
     >>>
     >>> subproc = h.LaunchSubProccess('pwd')
     >>> subproc.wait()
     0
     >>> subproc.communicate()[0]
     '/home/login/puppetmaster/src\n'
     >>>
You can get some information about your host and can launch foreground command or subprocess.Popen object.
